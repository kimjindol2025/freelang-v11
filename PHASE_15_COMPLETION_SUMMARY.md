**아키텍처 설계 및 문법 정의 완료**

```lisp

  SELECT u.*, COUNT(p.id) as post_count
```

```lisp
]
```


## 🔧 구현 상세

### 파서 확장
```typescript
// src/parser.ts — isBlockKeyword 확장
const isBlockKeyword = 
  ... || nextToken.type === T.Service 
      || nextToken.type === T.Controller 
      || nextToken.type === T.Guard 
      || nextToken.type === T.Model
      || nextToken.type === T.Query
      || nextToken.type === T.Migration
      || ...

// 및 parseBlock() 메서드 switch 추가
case T.Service: blockType = "SERVICE"; break;
case T.Controller: blockType = "CONTROLLER"; break;
case T.Guard: blockType = "GUARD"; break;
case T.Model: blockType = "MODEL"; break;
```

### 코드젠 확장
```typescript
// src/codegen-js.ts — genBlock() switch 추가
case "SERVICE": return this.genServiceBlock(node);
case "MODEL": return this.genModelBlock(node);
case "CONTROLLER": return this.genControllerBlock(node);
```

**생성된 메서드**:
- `genServiceBlock()` — TypeScript 클래스 생성
- `genModelBlock()` — SQL DDL + TS interface
- `genControllerBlock()` — Express 라우터 생성

### CLI 확장
```typescript
// src/cli.ts
function cmdCodegen(args: string[]): void { ... }

// switch 문에 추가
case "codegen": {
  cmdCodegen(args.slice(1));
  break;
}
```
