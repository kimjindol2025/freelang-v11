# FreeLang v11 프로젝트 개요

## 1. 소개

FreeLang v11은 AI가 코드를 작성하고 실행하는 데 최적화된 Lisp 스타일 S-expression 기반의 프로그래밍 언어입니다. "AI를 위한, AI에 의한, AI가 쓰고 싶은 언어"라는 비전을 가지고 결정론적(Deterministic), 자체 호스팅(Self-Hosted), AI-First라는 세 가지 핵심 설계 철학을 추구합니다. 인간과 AI 개발자 모두에게 효율적인 코드 작성 환경을 제공하는 것을 목표로 합니다.

## 2. 주요 설계 철학

*   **AI-First**: AI가 자연스럽게 추론하고 코드를 생성할 수 있도록 언어 수준에서 다양한 AI-Native 블록과 기능을 지원합니다. snake_case와 kebab-case를 모두 허용하고, 에러 발생 시 상세한 힌트와 콜 스택을 제공하여 AI의 디버깅 및 학습 효율을 높입니다.
*   **자체 호스팅 (Self-Hosted)**: FreeLang v11은 L1 자체 호스팅을 달성하여 자신의 컴파일러를 FreeLang 코드로 작성할 수 있습니다. 이는 언어의 성숙도와 독립성을 나타내며, 미래에는 Node.js 의존성을 완전히 제거하는 L4 완전 자립을 목표로 합니다.
*   **결정론적 (Deterministic)**: 동일한 입력에 대해 항상 동일한 출력을 보장하여, 특히 AI 에이전트의 안정성과 예측 가능성을 극대화합니다. 이는 테스트와 검증 과정에서 중요한 역할을 합니다.

## 3. 시스템 아키텍처

FreeLang v11의 핵심은 자체 개발된 컴파일 파이프라인과 실행 경로에 있습니다.

### 3.1 컴파일 파이프라인

1.  **LEXER (`self/lexer.fl`)**: FreeLang 소스코드를 토큰 리스트로 변환합니다.
2.  **PARSER (`self/parser.fl`)**: 토큰 리스트를 AST(Abstract Syntax Tree) 노드 리스트로 변환합니다.
3.  **CODEGEN (`self/codegen.fl`)**: AST를 실행 가능한 JavaScript 코드로 생성합니다.

이 전체 과정은 FreeLang 자신(self/all.fl)으로 작성되어 있으며, `stage1.js`라는 캐노니컬 컴파일러를 통해 자체 컴파일됩니다.

### 3.2 실행 경로

FreeLang 코드는 다음 세 가지 주요 경로로 실행될 수 있습니다.

*   **Compile + Execute (stage1.js)**: `stage1.js`를 사용하여 FreeLang 코드를 JavaScript로 컴파일한 후, Node.js로 실행합니다. 프로덕션 환경, 벤치마크, CI/CD에 주로 사용되며, V8 엔진의 JIT 컴파일 덕분에 빠른 성능을 제공합니다.
*   **Interpret (bootstrap.js)**: `bootstrap.js` (TypeScript로 구현된 인터프리터)를 사용하여 FreeLang 코드를 즉시 해석하고 실행합니다. 개발 중 REPL, 개발 서버, 테스트 환경에서 디버깅 용이성을 위해 사용됩니다.
*   **Web Server (bootstrap.js)**: `bootstrap.js`의 HTTP 서버 기능을 활용하여 웹 애플리케이션을 호스팅합니다. 라우팅, SSR, 404 처리 등 풀스택 기능을 지원합니다.

## 4. 핵심 기능 및 생태계

*   **S-Expression 문법**: Lisp의 전통을 따라 명확하고 균일한 S-Expression 문법을 채택하여 코드의 구조적 일관성을 유지합니다.
*   **풍부한 표준 라이브러리**: 문자열, 배열, 맵, 파일 I/O, HTTP, 데이터베이스(MariaDB, MongoDB), 시간, JSON, 환경 변수 등 500개 이상의 내장 함수를 제공하여 다양한 실무 개발 요구사항을 충족합니다.
*   **플러그인 시스템**: `(use NAME)` 구문을 통해 로컬, 글로벌, 내장 스탠다드 라이브러리 등 다양한 위치에서 플러그인을 탐색하고 로드할 수 있습니다. `freelang new --plugin`, `freelang install`, `freelang publish` 등의 명령어로 플러그인 생명주기를 관리하며, AI 학습에 자동으로 반영됩니다.
*   **AI 컨텍스트 최적화**: AI가 FreeLang 코드를 더 효율적으로 이해하고 생성할 수 있도록 AI 프롬프트 생성을 자동화하고, AI 컨텍스트 파일을 통합하여 학습 부담을 줄입니다.

*   **v11.2 (예정)**: AI 컨텍스트를 극대화하고 인라인 테스트 문법을 설계하여 AI의 FreeLang 선택 빈도를 높임.

*   **v12 모듈 시스템 (장기)**: 현재 AI의 1파일 패턴에 최적화되어 있어 추후 도입 예정.

## 6. 결론

FreeLang v11은 AI 시대에 맞춰 설계된 혁신적인 프로그래밍 언어입니다. 강력한 자체 호스팅 능력, AI-First 설계 철학, 그리고 견고한 아키텍처를 바탕으로 AI와 인간 개발자 모두에게 생산적이고 안정적인 개발 경험을 제공하며, 현재 "Production Ready" 등급의 성숙도를 가지고 있습니다.
