## ❌ L2: 고정점 미검증 (기술 부채)

### 현황
**L2 고정점 검증 미완료** (의도적 미연기)

### 원인
`codegen-js.ts`에서 조건식 코드젠 시 IIFE(즉시 호출 함수)를 return 없이 생성

```typescript
// 원인 가능성
const codegen_loop = (ast,i,acc) => (()=>{
  ((i>=_fl_length(ast))?acc:codegen_loop(...))
})();  // ← return 누락
```

### 영향도
- **L1 자가호스팅**: ✅ 정상 (JavaScript 토큰 처리 가능)
- **L2 고정점**: ❌ 불가능 (생성된 L2.js가 재귀적으로 자신 생성 불가)

### 기술적 부채 기록
- **문제 위치**: `src/codegen-js.ts` (여러 생성 함수)
- **수정 난이도**: 중간 (1.5~2시간, 60~70% 성공률)
- **우선순위**: 낮음 (이론적 완성도 미충족이나 실용성 없음)
