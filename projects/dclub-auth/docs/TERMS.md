# 이용약관 — dclub-auth

**시행일**: 2026-04-25
**문의**: support@dclub.kr

## 제1조 (목적)
본 약관은 dclub-auth(이하 "서비스")의 이용 조건과 운영자·이용자의 권리·의무를 정함.

## 제2조 (정의)
- "이용자": 본 서비스에 가입하거나 외부 IdP 로 로그인한 자연인
- "OIDC 클라이언트": 본 서비스를 인증 프로바이더로 등록한 외부 응용
- "Passkey": WebAuthn(W3C) 표준 기반 비밀번호 없는 인증 자격증명

## 제3조 (서비스 내용)
1. OpenID Connect 기반 인증·인가
2. 회원가입 / 로그인 / 비밀번호 재설정
3. 다중 인증 (TOTP, Passkey)
4. 외부 IdP 연동 (Google/Kakao/Naver)
5. 운영자 대시보드 (의심 로그인 감시)

## 제4조 (이용 자격)
- 만 14세 이상
- 정확한 정보 제공 의무
- 1인 1계정 (다중 계정 금지)

## 제5조 (계정 보안)
- 비밀번호 8자 이상, 본인 책임 보관
- MFA 활성 권장 (TOTP 또는 Passkey)
- 비밀번호 노출 의심 시 즉시 `/forgot` 재설정
- 의심 로그인 알림 메일 수신 시 즉시 본인 확인

## 제6조 (금지 행위)
- 타인 명의 도용·가입
- 자동화 도구로 가입·로그인 시도 (rate limit 트리거 시 15분 잠금)
- 서비스 무단 reverse-engineering 시도 (외부 인증 클라이언트 정상 통합은 권장)
- API 남용 (분당 30 요청 초과 시 차단)

## 제7조 (운영자의 의무·권리)
- 24시간 가용성 노력 (SLA 99% 목표, 공식 보장 아님)
- 침해 발견 시 즉시 영향 받은 사용자 통지
- 본 약관·정책 위반 시 사전 통지 후 계정 정지·삭제 가능

## 제8조 (서비스 변경·중단)
- 기능 추가·변경: 사전 공지
- 무중단 점검 원칙. 불가피한 중단 시 사전 공지 (1시간+ 점검은 24시간 전)
- 영구 종료 시 30일 전 공지 + 데이터 export 안내

## 제9조 (데이터 처리)
[개인정보처리방침](./PRIVACY.md) 참조.

## 제10조 (책임 한계)
- 천재지변·통신장애·외부 IdP(Google/Kakao 등) 장애로 인한 일시 불가용에 대해 책임 제한
- 본인의 비밀번호·디바이스(Passkey) 유실로 인한 손해는 본인 책임
- 보안 의무 다하지 못한 침해는 운영자 책임

## 제11조 (계정 삭제)
- 이용자는 언제든 `DELETE /me` 또는 계정 설정 페이지에서 탈퇴 가능
- 탈퇴 시 즉시 익명화 + 모든 인증 자료 삭제
- 탈퇴 후 같은 이메일로 재가입 가능

## 제12조 (분쟁 해결)
대한민국 법률 준거. 분쟁 시 서울중앙지방법원 관할.

## 제13조 (변경)
중요 변경 시 30일 전 공지 + 재동의 요청.
