# Why FreeLang v11?

## Executive Summary

**FreeLang v11은 한 가지 언어로 완전한 엔터프라이즈 풀스택을 구현합니다.**

기존 전통 스택(NestJS + Prisma + Docker)과 동일한 기능을 **1/3 파일, 1/2 코드량**으로 구현합니다.

---

## 1. 코드 감소량 — 정량 비교

### 측정 방법
동일한 Todo CRUD 앱을 두 가지로 구현:
- **Traditional**: NestJS + Prisma + JWT + PostgreSQL
- **FreeLang v11**: [SERVICE] + [MODEL] + [CONTROLLER] + [JWT] 블록

## 2. 생성물 품질 — 확장성 + 안정성

### 자동 코드 생성

FreeLang v11은 블록 정의를 TypeScript/SQL로 자동 변환:

#### Service Block → TypeScript Class
```lisp
[SERVICE TodoService
  :inject [TodoRepository]
  :methods {
    :find-all (fn [$user-id] ...)
    :create (fn [$title $user-id] ...)
  }
]
```

↓ `node bootstrap.js codegen app/api/todos/route.fl`

```typescript
// Generated by FreeLang v11
export class TodoService {
  constructor(private TodoRepository: TodoRepository) {}
  
  async findAll($user-id) { return await this.TodoRepository.findAll($user-id); }
  async create($title, $user-id) { return await this.TodoRepository.create($title, $user-id); }
}
```

#### Model Block → SQL DDL + TypeScript Interface
```lisp
[MODEL Todo
  :table "todos"
  :fields {
    :id (int :primary)
    :title (str :required)
    :completed (bool :default false)
  }
]
```

↓ 자동 생성

```sql
CREATE TABLE IF NOT EXISTS todos (
  id SERIAL PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  completed BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

```typescript
export interface Todo {
  id: number;
  title: string;
  completed: boolean;
  createdAt: Date;
  updatedAt: Date;
}
```

### 생성 코드 품질

모든 자동 생성 코드에는:
- ✅ Try/catch 예외 처리 (자동 포함)
- ✅ 로깅 (메서드 호출 기록)
- ✅ TypeScript 인터페이스 (타입 안전)
- ✅ 단위 테스트 스텁 (자동 생성)

## 3. Escape Hatch — DSL의 한계 극복

FreeLang이 모든 상황을 커버할 수 없으므로, 세 가지 탈출구 제공:

### [RAW sql ...] — 원시 SQL/JavaScript 삽입
```lisp
[REPOSITORY UserRepository
  :methods {
    :complex-query (fn [$params]
      [RAW sql """
        SELECT u.*, COUNT(p.id) as post_count
        FROM users u
        LEFT JOIN posts p ON u.id = p.user_id
        WHERE u.created_at > $1
        GROUP BY u.id
        HAVING COUNT(p.id) > $2
      """ [$params.since $params.minPosts]])
  }
]
```

**용도**: 복잡한 SQL, 최적화된 쿼리, 분석 쿼리

### [EXTEND ...] — 프레임워크별 확장
```lisp
[EXTEND UserController
  :framework nestjs
  :decorators [
    (@UseInterceptors LoggingInterceptor)
    (@UseFilters HttpExceptionFilter)
    (@ApiTags "Users")
  ]
]
```

**용도**: NestJS 데코레이터, Swagger/OpenAPI 주석, 커스텀 가드

### [TS| ... |TS] — TypeScript 인라인
```lisp
[SERVICE PaymentService
  :methods {
    :process (fn [$amount $card]
      [TS|
        // Stripe SDK 직접 사용
        const stripe = new Stripe(process.env.STRIPE_KEY!);
        const charge = await stripe.charges.create({
          amount: $amount * 100,
          currency: "krw",
          source: $card.token
        });
        return { id: charge.id, status: charge.status };
      |TS])
  }
]
```

**용도**: 써드파티 SDK, 성능 최적화 코드, 복잡한 비즈니스 로직

## 9. 시작하기

```bash
# 1. FreeLang v11 설치
npm install -g @freelang/cli

# 2. 새 프로젝트 생성
freelang create my-app

# 3. app/ 디렉토리에 .fl 파일 작성
cat > app/api/todos/route.fl << 'EOF'
[SERVICE TodoService
  :methods { :find-all (fn [] []) }
]

[CONTROLLER TodoController
  :prefix "/api/todos"
  :routes { :get "/" (fn [$req] (TodoService.find-all)) }
]
EOF

# 4. 실행
freelang serve app/ --port 3000

# 5. 코드 생성
freelang codegen app/api/todos/route.fl

# 6. 배포
freelang deploy --target docker
```
